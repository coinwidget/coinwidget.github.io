coinwidget

Bitcoin donation button generator

View on GitHub Download full.js Download simple.js

coinwidget helps you to receive donations in bitcoins. Just add the button to your website and start to receive donations!


<div id="coinwidget" data-icon="true" data-type="primary" data-wallet="1Gc317G5YdGm4Mx27RXpaa2vToFQR8J1bt" />
<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/ivandiazwm/coinwidget@0.1.1/builds/full.js" charset="utf-8"></script>

Get by CDN

coinwidget is very simple to use, just download the file and apply it as it was done in the example above

You can retrieve the files in different ways, the recomended is to use jsdeliver https://cdn.jsdelivr.net/gh/ivandiazwm/coinwidget@0.1.1/builds/full.js

If you already have jquery and semantic-ui imported, you can use the simple version (lightweight) https://cdn.jsdelivr.net/gh/ivandiazwm/coinwidget@0.1.1/builds/simple.js

Usage

Just right before importing the script, create a div with id="coinwidget". You can use the fallowing addition data-* attributes:

  • data-wallet: put the address of your bitcoin wallet (required)
  • data-type: put the type of the button
  • data-icon: put true if you want the icon shown in the button
  • data-text: put an alternative text to the button
